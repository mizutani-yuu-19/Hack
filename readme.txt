作品名	Hack
制作年月	2018年12月
製作期間	６日
個人製作

ゲーム概要
Hackは３Dシューティングゲームです。
自機のライフは３。３回ダメージを受ける前に
そのステージにいる敵をすべて倒せばクリアです。
ステージは全５面です。
クリアしたステージをプレイすると難易度が上がっています。
難易度は４段階。
友人に遊んでもらったところ、もう一段階難易度が高いものを遊びたいと言われたので、
Lv.5を作成しました。

アピールポイント
こちらの作品はC++のクラスをしっかりと使って作ることができました。
ニーアオートマタに出てくるハッキングを参考に作りましたが、
本家ほどのステージを作る時間がなかったので難易度を上げるという方法でボリュームを増やしました。
グループワークの進級作品を作っている中で作成したので制作期間は６日です。
モデルやエフェクトも自分で作りました。すべて自分で制作したのは初めてです。
他のDirectXの作品に比べプログラムの構成がきれいに作成できたと思います。
Lv.５を作成する際は引数を変えるだけで難易度調整できるようにしていたので簡単にできました。
Lv.６以上にすると私でも攻略できませんでしたので、５までしか作っていません。
